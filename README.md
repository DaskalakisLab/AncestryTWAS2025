# AncestryTWAS2025
This repository contains template scripts used in our paper titled: "Leveraging genomic and transcriptomic data of diverse ancestry to uncover mechanisms of psychiatric risk in the adult and developing brain" (AncestryTWAS2025) at Nature Communications. 

To access GReX models visit our zenodo folder [10.5281/zenodo.17682742](https://zenodo.org/records/17682743).
